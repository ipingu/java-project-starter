Basic how-to

mvn jetty:run : run an embedded jetty server where the webapp is deployed

mvn dependency:copy-dependencies : copy project dependencies into a local repository

Tomcat in debug mode : Add this JVM options "-Xdebug -Xrunjdwp:transport=dt_socket,address=8000,server=y,suspend=n", or Set environment variables JPDA_ADDRESS=8000 and JPDA_TRANSPORT=dt_socket and then start tomcat using catalina jpda start.
